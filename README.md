# Data Job Analysis

Application d'analyse rapide des dernieres offres d'emploi dans le secteur de la data en France.

## Objectif

Fournir une vue synthetique et actualisee du marche de l'emploi data en France :

- **Competences demandees** : langages, outils, frameworks les plus recherches
- **Mode de travail** : remote, hybride, presentiel
- **Salaires** : fourchettes par poste, experience, localisation
- **Types de postes** : Data Analyst, Data Engineer, Data Scientist, ML Engineer, etc.
- **Localisations** : repartition geographique des offres
